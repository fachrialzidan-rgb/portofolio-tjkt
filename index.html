Tentang Saya

Saya adalah siswa SMK Wikrama Bogor jurusan Teknik Jaringan Komputer dan Telekomunikasi (TJKT). Saya memiliki minat besar dalam bidang jaringan komputer, administrasi server, cloud computing, dan keamanan jaringan.

Saya terbiasa mengonfigurasi MikroTik untuk hotspot, routing, firewall, dan manajemen bandwidth, membangun server berbasis Linux (DNS, DHCP, Web Server), serta melakukan deployment aplikasi di Cloud menggunakan AWS (EC2, Load Balancer, Auto Scaling). Saya juga terbiasa memantau kondisi jaringan dan server menggunakan Zabbix.

Saya terbuka untuk kesempatan magang industri maupun posisi entry-level di bidang Network Support, System Administrator, atau Cloud Support.

Proyek & Praktik

Konfigurasi MikroTik

MikroTik Hotspot Firewall

Linux Server

Instalasi dan konfigurasi Ubuntu Server beserta layanan DNS Server, DHCP Server, dan Web Server untuk kebutuhan jaringan internal.

Ubuntu Server DNS/DHCP Web Server

AWS Cloud

Melakukan deployment website menggunakan layanan Amazon EC2, mengatur Load Balancer untuk distribusi trafik, dan menerapkan Auto Scaling agar server tetap stabil saat trafik meningkat.

AWS EC2 Load Balancer Auto Scaling

Zabbix Monitoring

Memantau status server dan perangkat jaringan secara real-time menggunakan Zabbix, termasuk pengaturan alert untuk mendeteksi gangguan lebih awal.

Zabbix Monitoring Alerting

Konfigurasi VLAN & Routing Antar Departemen

Cisco Packet Tracer VLAN Routing
